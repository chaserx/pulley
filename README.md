# Pulley

Command line application for updating all git repositories in a directory.

# AUTHOR'S NOTE

In hindsight, I can see why you might not want to automate this process. I'll leave this here for now as a guidepost.

## Installation

    $ gem install pulley

## Usage

pulley ~/code
